=========================================
Day 3 - Overly complex Pomodoro Timer
=========================================
In this module i practiced using the *datetime* module.
Instead of using the *time* module to create the Pomodoro Timer, i am using *datetime* features.

I chose to use a class for the Pomodoro Timer, to easily control the state of the timer such as starting and pausing.
If the user want to reset the timer, i just create a new instance of the class.

In the .. code-block:: main() .. i built a user interface for using the timer.
I practiced using threading here to be able to run the timer simultaneously while reading user input.
